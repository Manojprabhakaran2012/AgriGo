# AgriGo

AgriGo is a Flask-based web application for crop recommendation, fertilizer suggestion, and crop disease detection using ML/DL models.

## Features
- Crop recommendation based on soil and weather
- Fertilizer recommendation
- Crop disease detection from images
- Uses TensorFlow and Keras models
- Interactive front-end with HTML, CSS, and JS

## Installation
1. Clone the repo
2. Create a virtual environment: `python -m venv venv`
3. Activate venv: `venv\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run: `python app.py`
